The datasets consists of branches of the National Bank of Algeria (BNA)


## Source
[BNA](https://www.bna.dz/fr/).

 

## Data format

| Key          | Description | 
| ------------ | -----------------------|
| nom       |Branch name |
| email     |Branch email                 |
| adresse |Address of the branch |
| code_wilay| Wilaya code of where the branch is located |
| wilaya |Wilaya name                |
| tel | Telephone number(s)                 |
| fax | Fax number(s) |
| long | Longitude                 |
| lat |Latitude            |
