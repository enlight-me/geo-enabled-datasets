## Dataset description 
The raw data is available as [`ndjson`](http://ndjson.org/) files seperated by category, in the following format: 

| Key          | Description | 
| ------------ | -----------------------|
| nom       |Branch name |
| email     |Branch email                 |
| adresse |Address of the branch |
| code_wilay| Wilaya code of where the branch is located |
| Wilaya |Wilaya name                |
| long | Longitude                 |
| lat |Latitude            |
